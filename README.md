# create2LocalController #

### What is this repository for? 
To control iRobot Create2 with xbox controller using Node.js

### How do I get set up? ###

```sh
$ git clone https://github.com/cdra0624/create2LocalController.git
$ cd create2LocalController
$ npm install
$ npm start
```
